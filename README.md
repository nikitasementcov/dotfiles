# Dotfiles

1. Clone that repo (~/dotfiles is preffered dir to use stow without extra arguments)
2. Install GNU stow (could be installed by `brew`)
3. Use GNU stow to create symlinks to the configs.
    `stow nvim alacritty tmux`
